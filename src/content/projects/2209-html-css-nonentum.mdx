**기간**: Sep 2022 (1 day)


**역할**: HTML/CSS 클론 코딩 학습


🔗 코드 보기: [GitHub Repository](https://github.com/iam-sunny/web-basic-clones/tree/main/nonentum)


## 1. **프로젝트 소개**


이 프로젝트는 **Nonentum 웹사이트 레이아웃을 클론 코딩**하면서 HTML과 CSS의 기초를 다지는 것을 목표로 했습니다.


순수 HTML과 CSS만으로 웹사이트 구조를 모방하여, **시멘틱 마크업·레이아웃 구성·스타일링**에 대한 이해를 높였습니다.


![image1.png](/assets/notion/26a10b9ff726807a92f5d0fb9e111916/78154487-image1.png)


## 2. **학습 포인트**

- **HTML 구조 설계**
    - header, nav, main, footer 등 시멘틱 태그 사용
    - 콘텐츠 구획화와 재사용 가능한 구조 연습
- **CSS 레이아웃 실습**
    - Flexbox를 활용한 정렬 및 배치
    - Grid를 통한 영역 분할
    - position 속성으로 요소 배치 제어
- **디자인 디테일 학습**
    - 폰트 크기, 색상 팔레트, 여백 조정
    - hover, active 등 인터랙션 스타일링

💻 _[코드 스니펫 예시 — Grid 레이아웃]_


```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 3fr;
  gap: 20px;
}

.sidebar {
  background: #f9f9f9;
}

.content {
  background: #fff;
}
```


## 3. **성과 및 배운 점**

- **웹사이트 클론 경험**을 통해 HTML/CSS 문법과 레이아웃 설계에 자신감 획득
- Flexbox와 Grid를 반복적으로 활용하면서 두 방식의 장단점을 체감
- 단순히 화면을 “그럴싸하게” 만드는 게 아니라, **구조적이고 유지보수 가능한 마크업**이 중요함을 학습
- 작은 차이(폰트 크기, 정렬, 색상 톤 등)가 사이트 완성도에 큰 영향을 준다는 점 인식

![image2.png](/assets/notion/26a10b9ff726807a92f5d0fb9e111916/47004d9e-image2.png)


## 4. **다음 학습 계획**

- 이번에는 HTML/CSS에 집중했으므로, 다음 단계는 **JavaScript DOM 조작**과 간단한 이벤트 처리 학습
- 반응형 웹(Responsive Web)과 미디어 쿼리를 적용하여 다양한 화면 크기에 대응
- 차후에는 React/Next.js 환경에서 같은 UI를 다시 만들어보고, 컴포넌트 기반 설계로 확장

💻 _[코드 스니펫 예시 — 미디어 쿼리]_


```css
@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}
```


## 5. **정리**


Nonentum Clone은 포트폴리오 작품이라기보다, **웹 개발 기초 학습 과정**을 기록한 스터디 프로젝트입니다.


실습을 통해 HTML/CSS 기본기를 다졌고, 향후 JavaScript 및 프레임워크 학습으로 넘어가기 위한 기초를 마련했습니다.
